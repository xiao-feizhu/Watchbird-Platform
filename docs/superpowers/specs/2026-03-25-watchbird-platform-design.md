## 1. 项目概述

### 1.1 产品定位

观鸟资源整合平台是一个 C2C 服务型电商平台，连接商业观鸟机构/独立鸟导与观鸟爱好者，解决鸟导客源问题，同时为爱好者提供专业、可靠的观鸟服务。

### 1.2 核心价值主张

- **对鸟导/机构**: 持续稳定的客源渠道，分级认证体系提升信誉
- **对观鸟爱好者**: 专业可靠的观鸟服务，评价系统降低选择风险
- **平台差异化**: 垂直专业（鸟讯/鸟点数据）+ 社区氛围 + 交易保障

### 1.3 商业模式

| 要素 | 说明 |
|------|------|
| 盈利模式 | 平台担保交易抽成（阶梯费率） |
| 费率结构 | 新手5% → 资深8% → 金牌12% → 顶级15%，大额订单享优惠 |
| 资金流 | 用户付款 → 平台托管 → 服务完成 → 结算给鸟导 |
| 认证体系 | 分级认证（基础/高级/金牌/顶级），不同费率与曝光权益 |

## 2. 用户角色与权限

### 2.1 角色定义

| 角色 | 定义 | 核心权限 |
|------|------|----------|
| **观鸟爱好者** | 普通用户，寻求观鸟服务 | 浏览、搜索、下单、评价、申诉 |
| **鸟导** | 独立从业者，提供观鸟向导服务 | 发布服务、接单、带队、查看评价 |
| **机构** | 商业观鸟机构，管理多名鸟导 | 批量发布、子账号管理、数据看板 |
| **政府/景区/环保** | 官方机构，发布公益/官方活动 | 合作专区入驻、发布活动、志愿者招募、景区推广 |
| **平台运营** | 官方管理员 | 审核、仲裁、配置、数据监控 |

### 2.3 合作专区（政府/景区/环保机构）

政府背景机构以独立专区形式入驻，区别于商业 C2C 交易。

#### 专区定位
- **性质**: 官方合作背书，公益与商业结合
- **展示**: 独立入口 "官方合作" / "公益观鸟"
- **信任**: 政府认证标识，提升公信力

#### 入驻类型

| 类型 | 主体 | 典型场景 |
|------|------|----------|
| **景区** | 国家公园、湿地公园、自然保护区 | 观鸟旅游线路推广、生态体验预约 |
| **环保机构** | 观鸟协会、环保 NGO、科研机构 | 公民科学项目、志愿者招募、科普活动 |
| **政府部门** | 林业局、文旅局、教育局 | 观鸟节、科普周、研学活动 |

#### 活动模式

| 模式 | 资金方式 | 平台角色 | 示例 |
|------|----------|----------|------|
| **公益免费** | 政府/机构全额承担 | 报名管理、志愿者管理 | 湿地保护志愿者招募 |
| **有偿服务** | 用户付费参与 | 交易撮合、资金托管 | 景区观鸟导赏团 |
| **补贴模式** | 政府补贴 + 用户自费 | 补贴发放、价格展示 | 市民观鸟体验券（原价 200，补贴后 50） |

#### 与 C2C 的差异

| 维度 | C2C 交易 | 政府合作专区 |
|------|----------|--------------|
| 费率 | 阶梯抽成 5%-15% | 免抽成或固定服务费 |
| 审核 | 平台运营审核 | 政府关系部门对接 |
| 展示 | 按算法排序 | 专区置顶 + 活动日历 |
| 合同 | 平台标准协议 | 单独合作协议 |
| 结算 | 自动结算到钱包 | 对公转账/财政系统对接 |

## 5. 评价与审核机制

### 5.1 双边评价流程

1. **服务完成后** → 双方进入待评价状态
2. **用户评价鸟导**: 评分(1-5星) + 文字 + 标签
3. **鸟导评价用户**: 评分(1-5星) + 文字
4. **平台审核**: 低分（≤3星）或包含敏感词的评价触发人工审核
5. **评价展示**: 审核通过后对外展示

### 5.2 平台介入触发条件

| 触发条件 | 处理方式 |
|----------|----------|
| 评分 ≤ 3星 | 自动标记，运营24小时内审核 |
| 用户/鸟导申诉 | 运营介入调查，48小时内仲裁 |
| 包含敏感词 | 先隐藏，人工审核后决定 |
| 被多次举报 | 暂停展示，深度调查 |

### 5.3 评价权重算法

```
综合评分 = Σ(评分 × 权重) / Σ权重

权重计算：
- 基础权重: 1
- 有文字评价: +0.5
- 有图片: +0.3
- 认证用户: +0.2
- 订单金额高: +0.2
```

## 6. 认证与等级体系

### 6.1 认证级别

| 级别 | 要求 | 权益 |
|------|------|------|
| **基础** | 身份证 + 手机号 + 基础培训 | 平台抽成15%，基础曝光 |
| **高级** | 基础 + 从业证明/导游证 | 平台抽成12%，优先曝光 |
| **金牌** | 高级 + 50单好评率>95% | 平台抽成8%，置顶推荐 |
| **顶级** | 金牌 + 100单好评率>98% + 平台面试 | 平台抽成5%，首页展示 |

### 6.2 认证流程

```
提交资料 ──> 系统初审 ──> 人工复审 ──> 认证结果
                │              │
                │              └── 通过: 发放等级标识
                │              └── 拒绝: 说明原因，可重新申请
                └── 自动校验身份证/手机号真实性
```

## 8. 关键业务流程

### 8.1 下单支付流程

```
1. 用户选择服务 → 选择日期 → 填写人数
2. 系统生成订单 → 状态: 待付款
3. 调起微信支付 → 用户完成支付
4. 微信支付回调 → 更新订单状态: 待确认
5. 通知鸟导确认
```

### 8.2 资金结算流程

```
服务完成 + 评价完成 → 订单状态: 已完成
                        ↓
               系统计算应结算金额
               （订单金额 - 平台抽成）
                        ↓
               资金从平台账户划转至鸟导钱包
                        ↓
               鸟导申请提现 → 打款至绑定银行卡
```

### 8.3 申诉仲裁流程

```
用户/鸟导提交申诉 → 上传证据 + 描述问题
                          ↓
              运营后台接收申诉工单
                          ↓
              48小时内调查取证
                          ↓
              仲裁结果:
              ├─ 支持用户 → 全额/部分退款
              ├─ 支持鸟导 → 订单正常结算
              └─ 双方有责 → 协商解决
```

### 8.4 政府合作专区业务流程

```
机构入驻申请 ──> 商务对接/资质审核 ──> 签署合作协议 ──> 开通机构账号
                                                    │
                                                    └── 发布活动
                                                          │
                            ┌─────────────────────────────┼─────────────────────────────┐
                            │                             │                             │
                            ▼                             ▼                             ▼
                     ┌──────────────┐           ┌──────────────┐           ┌──────────────┐
                     │  公益免费    │           │  有偿服务    │           │  补贴模式    │
                     └──────┬───────┘           └──────┬───────┘           └──────┬───────┘
                            │                          │                          │
                            ▼                          ▼                          ▼
                     志愿者报名管理             用户下单支付              补贴资格审核
                            │                          │                          │
                            ▼                          ▼                          ▼
                     名单确认/通知               服务履约                  补贴后价格展示
                            │                          │                          │
                            ▼                          ▼                          ▼
                     活动执行/签到               评价/结算                 平台结算+补贴对账
```

### 8.5 补贴模式资金流

```
用户支付（补贴后价格） ──> 平台托管
                                   │
              ┌────────────────────┼────────────────────┐
              │                    │                    │
              ▼                    ▼                    ▼
        鸟导结算              平台抽成              政府补贴账户
        （服务费用）          （免抽成或固定费）     （按协议结算）
```

### 9.1 安全措施

| 层面 | 措施 |
|------|------|
| 数据传输 | 全站 HTTPS，TLS 1.3 |
| 身份认证 | JWT + Refresh Token，敏感操作二次验证 |
| 支付安全 | 微信支付签名验证，防重放攻击 |
| 数据存储 | 身份证等敏感信息 AES-256 加密 |
| 接口防护 | 速率限制，防爆破，防SQL注入 |

### 9.2 风控策略

| 风险 | 策略 |
|------|------|
| 恶意刷单 | 同一设备/手机号频率限制，异常订单标记 |
| 洗钱风险 | 大额订单人工审核，提现频次限制 |
| 虚假评价 | 评价者与订单关联验证，文本相似度检测 |
| 账号盗用 | 异地登录提醒，敏感操作短信验证 |

## 11. 运营规划

1. **种子鸟导招募**: 联系10-20位知名鸟导，给予顶级认证 + 免抽成优惠
